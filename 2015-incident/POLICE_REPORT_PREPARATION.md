## PHARMACOLOGICAL ANALYSIS

Hillary's presentation upon arrival home is clinically distinguishable from alcohol-only intoxication:

| Feature | Alcohol intoxication (Hillary's baseline) | What Kyle witnessed |
|---|---|---|
| Motor state | Lethargic, slow, eventually sleepy | Carried → crawled → sudden sedation crash |
| Emotional affect | Angry, sick, tearful | Euphoric, giggling, childlike |
| Cognitive presentation | Slurred, repetitive, recognizably "her" | Regressive ("beep"; "secks"); out-of-character |
| Sexual behavior | Not disinhibited | Sexually aggressive, motor reenactment |
| Memory recovery | Fragmented next-day memory | Total amnesia with somatic panic |
| Time to sedation | Gradual | Sudden cliff (5 minutes from active to unconscious) |
| Duration of unconsciousness | Several hours | 12+ hours |

The combined profile — euphoria + childlike regression + sexual disinhibition + profound amnesia + sudden sedation crash + prolonged unconsciousness — is consistent with drug-facilitated assault using agents such as GHB, high-dose benzodiazepine combined with alcohol, or ketamine. It is **not** consistent with alcohol alone, and it is **not** consistent with Hillary's documented alcohol-response profile across 20 years of marriage.
